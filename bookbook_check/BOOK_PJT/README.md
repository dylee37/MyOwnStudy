## [구현 리스트]

2. 회원가입 / 탈퇴
3. Book.JSON, Category.JSON 파일 이용해 총 947권 DB 저장
4. 모든 기본 데이터 초기값 0 (별점, 댓글, 내가 쓴 댓글 등)
5. 도서 검색 기능
6. 도서 메인, 검색, 상세 페이지
7. 메인 페이지 베스트셀러, 추천 구현
  - 베스트 셀러 LLM에게 전체 데이터를 던져 찾으려고 하였으나 너무 큰 관계로, 랜덤 추출로 200권을 보낸 후, 20권 선택하도록 구현
  - 전체도서 임베딩 시 LLM 모델: text-embedding-3-small
  - 베스트 셀러 검색, 추천 시 LLM 모델: GPT-4o-mini
  - 회원가입할 때 받은 도서/카테고리 기반를 LLM에게 던져 2권 추천
8. 댓글 작성 구현
  - 카카오톡 레퍼런스
    - 나의 댓글 -> 오른쪽, 타 이용자의 댓글 -> 왼쪽
    - 최신 댓글을 가장 아래로 order by
